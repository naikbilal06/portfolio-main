# Portfolio Website — Bilal Ahmad Naikoo

A professional portfolio website for Bilal Ahmad Naikoo, built with Next.js, React, Tailwind CSS, and Framer Motion.

The site showcases:

- a hero section with animated introductions
- an about section with resume-aligned experience and education
- a technical skills section
- featured projects with detailed descriptions
- a contact section and message form
- a dedicated resume page at `/resume`

## Live Preview

Open the app locally after starting the development server:

```bash
npm run dev
```

Then visit:

- `http://localhost:3000`
- `http://localhost:3000/resume`

## Project Structure

- `src/app/page.tsx` — main landing page
- `src/app/components` — reusable page sections
- `src/app/resume/page.tsx` — dedicated resume page
- `src/app/data` — project and skill data
- `public/resume.html` — downloadable resume file

## Installation

Install dependencies first:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

## Build for Production

Build the project:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
